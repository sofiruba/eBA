# eBA

eBA es una aplicación pensada para conectar personas que quieren asistir a eventos pero no tienen con quién ir. La propuesta se centra en que el usuario pueda descubrir eventos, ver quiénes también están interesados y conectar con otras personas antes de asistir.

## Objetivo del proyecto

El objetivo del MVP es validar el flujo principal de la aplicación:

Registro / Login  
→ Descubrir eventos  
→ Ver detalle de un evento  
→ Ver personas interesadas  
→ Conectar con otros usuarios
